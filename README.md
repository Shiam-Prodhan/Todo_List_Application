# TodoListApplication

A simple and responsive Todo List Application built using **AngularJS** that demonstrates complete **CRUD (Create, Read, Update, Delete)** operations. This project helps users manage daily tasks efficiently through an intuitive user interface.

## Features

- ✅ Add new tasks
- 📋 View all tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 🔄 Real-time data binding using AngularJS
- 📱 Responsive and user-friendly interface

## Technologies Used

- AngularJS
- HTML5
- CSS3
- JavaScript
- Bootstrap (optional)

## CRUD Functionalities

### Create
Users can add new tasks to the todo list.

### Read
Displays all available tasks in a structured list.

### Update
Existing tasks can be modified or marked as completed.

### Delete
Users can remove tasks that are no longer needed.

## Installation

1. Clone the repository:

```bash
git clone [https://github.com/your-username/TodoListApplication.git](https://github.com/Shiam-Prodhan/Todo_List_Application)
```

2. Navigate to the project directory:

```bash
cd TodoListApplication
```

3. Open `index.html` in your browser or run it using a local server.

## Usage

1. Enter a task in the input field.
2. Click the **Add** button to create a new task.
3. Edit tasks using the update option.
4. Delete tasks when they are no longer required.
5. Manage your daily activities efficiently.
